Contexte : 

 

Dans le cadre de son développement et de la gestion de ses ressources humaines, la conception d'un logiciel de gestion de mots de passe personnalisé, nommé "SUPWARDEN", est prévue comme alternative aux solutions existantes telles que Keepass, Bitwarden, Dashlane, etc., qui sont parfois payantes. 

 

Fonctionnalités : 

Permettre aux employés de créer, administrer et partager des identifiants et mots de passe. 

Chaque employé possède un compte personnel avec un trousseau de mots de passe personnel. 

Possibilité de créer des trousseaux partagés pour que plusieurs membres puissent consulter ou créer des mots de passe communs. 

Possibilité pour chaque membre de définir qui peut modifier les mots de passe dans le trousseau partagé (soit tout le groupe, soit certains membres spécifiques). 

 

Environnement technique :  

C# 

.Net Core 

Entity Framework Core 

ASP .NET Identity 

API Rest 

Microsft SQL server 

MVC Razor 

Docker
